# About
This is a simple implementation of a sketching inrerface, where an empty square canvas that when a pixle is hovered over with a mouse changes colors randomly

## Objective
In this project I will attempt to create the "Etch-a-Sketch" game, using javascript. By the end of this project, I hope to be able to demonstrate better problem solving skills and a better understanding of the syntax of javascript and Css.

## Built With

* HTML
* CSS
* Javascript
